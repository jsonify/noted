# Step 1: Open Template Browser

The **Template Browser** is your visual hub for managing all templates with custom variables.

## What You'll See

When you open the Template Browser, you'll find:

- **Grid/List Views**: Toggle between visual cards or compact list
- **Search Bar**: Filter templates by name, description, or tags
- **Category Filters**: Browse by Built-in, Custom, or specific categories
- **Template Cards**: Each showing name, description, category, tags, and usage count

## Your First Action

Click the **"Create New Template"** button to start building a template with custom variables.

## Pro Tip

Use the search bar to quickly find templates. Try searching for "bug" or "meeting" to see examples of templates with variables!
